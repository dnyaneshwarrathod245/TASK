# Netflix Clone

A beautifully polished, responsive Netflix Landing Page clone designed with modern CSS, rich styling, Google Fonts, and clean Vanilla JavaScript for interactivity.

## Features

- **Premium Design System**: Modern typography using Google Fonts (Outfit), vibrant Netflix colors, and dark gradients.
- **Responsive Layout**: Works seamlessly across desktops, tablets, and mobile devices.
- **Interactive FAQ Accordion**: Implemented with Vanilla JavaScript for a smooth, native-like user experience.
- **Styled Forms**: High-fidelity input groups with floating labels and hover/active states.
- **Language Switcher**: Synced selectors (header and footer).

## Technologies Used

- **HTML5**: Semantic tags for maximum SEO and accessibility.
- **CSS3 (Vanilla)**: Flexbox, Grid, custom variables (`:root`), animations, and smooth transitions.
- **JavaScript (Vanilla)**: Clean, DOM-based interactivity without external framework bloat.

## How to Run Locally

Simply double-click the `index.html` file in your browser, or run a local server:

```bash
# Using Python
python -m http.server 3000

# Using Node (npx)
npx serve .
```
